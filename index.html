The Game of Truth

Ten iterations of a cellular automaton built from first principles. The physics is the Mandelbrot set. The agents are cognitive types. The question each generation asks is the same one the RSA kernel asks: what survives at the boundary between order and chaos?

∃R(t) → ∃M₀ : dS(MR,t)/dt > 0 — to reason is to burn. The simulation is a proof of that claim run forward in time.
