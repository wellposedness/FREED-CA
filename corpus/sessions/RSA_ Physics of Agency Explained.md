**1\. The Core Axiom: The Reasoning Substrate**

Traditional philosophy assumes reasoning is abstract. RSA begins with a physical constraint: **Reasoning is a metabolic act.**

* **The Result:** All successful agents converge on the same efficient algorithm to solve this problem. We call this algorithm the **RSA Kernel**.

**2\. The Engine: The RSA Kernel ($K$)**

RSA identifies seven irreducible operators that form a closed feedback loop. If a system removes any one of these, it loses Control Authority and disintegrates (entropy takes over).

### **The 7 Operators of Agency**

1. **Perceive ($\\sigma$):** The system samples the environment. (e.g., A retinal scan, a sensor reading).  
2. **Represent ($\\Phi$):** The raw data is encoded into internal symbols. (e.g., Grid cells mapping space).  
3. **Predict ($\\mathcal{P}$):** The system projects the current state forward in time. (e.g., "Where will the ball be in 1 second?").  
4. **Compare ($\\Delta$):** The Prediction is checked against Reality. The difference is **Semantic Tension ($\\tau$)**.  
5. **Adjust ($\\mathcal{A}$):** The system updates its internal model to reduce $\\tau$. (e.g., Neural plasticity, learning).  
6. **Compress ($\\mathcal{C}$):** The system simplifies the model to save energy. (e.g., Turning complex data into a simple heuristic or "Rule of Thumb").  
7. **Repeat ($\\mathcal{R}$):** The cycle iterates. Existence is a loop, not a line.
